The MultiLang extension allows you to tell Search Engines where your macrolinguistic websites are located.

== Description ==

***When you have websites in other languages, without the use of a lingistic plugin, this extension is a must-have to activate.***

This extension adds extra options within the "Advanced Settings" submenu group.

= Compatibility =

* This plugin requires **The SEO Framework 2.7.0 and up**.
* This plugin works on multisite.
* This plugin should not be used together with PolyLang, WPML and qTranslate X.
	* Those plugins output the inserted tags themselves.

== Installation ==

1. Activate extension.
2. Fill in options in the advanced tab.
3. Save.

== Frequently Asked Questions ==

= What is this? =

This free extension is made to help Google direct visitors to the correct website.

= How does this plugin work? =

This plugin lets you fill in the website addresses located elsewhere with their correct language tag.

= Does this plugin work on multisite? =

Absolutely!
